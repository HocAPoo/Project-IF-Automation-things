Blend Those Ideas Together

I first started learning Blender in 2020, at the height of the Covid-19 pandemic. I stumbled upon Blender by chance from a YouTube video titled "Why Blender is Hard" I started with basic low-poly modeling and later transitioned to using Blender to model 3D prints, this way I would be able to practice modeling more.

The animation that you see at the top of the webpage was created and animated in Blender! Nowadays I like to print one-to-one scale objects from video games.
